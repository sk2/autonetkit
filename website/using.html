Using


Using AutoNetkit.

Note: this section assumes you have installed the "Development Alpha" using the above instructions.

Drawing a Network

  1. Download yEd from yWorks.
  2. Clicking will create a node. By default nodes are treated as routers.
  3. To create a link between nodes, click the mouse on the source node, and release on the destination node. AutoNetkit will ignore directional edges, so don't worry about the arrows. An example network is shown below:
    4. .
  4. Save the diagram in graphml format, eg test.graphml.
  5. Run autonetkit with this input file autonetkit -f test.graphml. You should receive output similar to the following: 
    sk:Desktop sk2$ autonetkit -f test.graphml 
INFO AutoNetkit 0.0.34
INFO Compiling Netkit for localhost
INFO Rendering Network

Setting up Webserver for Visualisation

  1. You will need to start the webserver for AutoNetkit to connect to. This can be run as the normal user (sudo is not required). 
    $ ank_webserver
    This will start the webserver on port 8000. You can choose a different port such as 8080: 
    $ ank_webserver 8080
    however this will require a configuration file update to specify the alternate port.
  2. You should now be able to connect in a web browser to the site localhost:8000. A modern browser such as Chrome, Safari or Firefox should work fine. The visualisations has been tested most in Google Chrome. This will display a placeholder network topology.
  3. You can now run AutoNetkit, which will send the topology automatically to the webserver. Run autonetkit as before: 
    autonetkit -f test.graphml
    You should see the webpage update to reflect your network topology.
  4. It is often convenient to put autonetkit into monitor mode, where it will automatically re-generate the configuration files, and update the webserver, if the input file changes. This can be done using the --monitor parameter: 
    autonetkit -f test.graphml --monitor
    You should now be able to update the network diagram, with the visualisation automatically updated.
